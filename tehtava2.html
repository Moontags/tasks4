Attaching Event Handlers in JavaScript

Different Ways to Attach Event Handlers:

1. HTML Attributes

Attach handler directly in HTML.

<button onclick="console.log('Pressed!')">Press Here</button>

Pros: Simple for basic actions.

Cons: Mixes HTML/JS, limited logic, one handler only, confusing this.

2. DOM Element Properties

Set handler as a JS property.


const button1 = document.getElementById('button1');
button1.onclick = function() {
    console.log('Pressed via property!');
};

Pros: Clearer separation than HTML attributes.

Cons: One handler only, overwrites existing.

3. addEventListener() Method

Recommended method for attaching handlers.


const button2 = document.getElementById('button2');
button2.addEventListener('click', function() {
    console.log('Pressed via listener!');
});

button2.addEventListener('mouseover', function() {
    console.log('Mouse over!');
});

Pros: Multiple handlers, better control (removeEventListener), clear syntax, works with various objects, controls capturing/bubbling, predictable this.

Cons: Slightly longer syntax.

In summary, addEventListener() is generally the best and most versatile way.
